HTML

Hypertext Markup Language (HTML)

태그 (Tag)

<>로 태그를 시작하고 로 닫아준다
글씨 진하게 : strong
밑줄(underline) : u
제목(heading) : h1~6, 숫자가 커질수록 글자가 작아짐
줄바꿈 : br, 닫는 태그 X
단락(paragraph) : p

br과 p의 차이 = p는 단락 사이의 간격 고정으로 시각적 자유도가 떨어짐
=> br은 쓰는 만큼 줄바꿈 가능, 많은 사람들이 선호
css 이용 시 p 태그의 한계 극복 가능
p 태그를 통해 단락의 경계를 분명히, CSS를 통해 p 태그의 디자인 변경 가능. 따라서 br보다 p가 더 좋은 선택

이미지(image) 넣기 : img, 닫는 태그 X

목록 표현 (list) : li
경계 : ul(unordered list) / ol(ordered list) 순서/숫자 유무의 차이
li 태그는 ul 태그를 꼭 필요로 함, ul 태그 역시 li가 없다면 존재 가치 X => 부모 자식 관계 제목 : title
웹브라우저의 문자 표현 해석 지정 : meta charset = ""
본문 : body
본문을 설명하는 태그 : head
웹페이지가 HTML로 만들어짐을 표현 : !doctype html, html로 닫음
링크 걸기 (anchor) : a

속성 (attribute)

속성 = 태그의 심화된 문법
이미지 태그에서 src가 속성, 속성의 값 = 이미지의 주소
이미지 크기 조정 : width href(HyperText Reference) : 링크된 페이지의 URL 명시
새 창에서 페이지 열기 : target = "_blank"
title : 링크가 어떤 내용을 담고 있는지를 툴팁으로 보여줌 = 마우스 갖다대면 나오는 설명창
